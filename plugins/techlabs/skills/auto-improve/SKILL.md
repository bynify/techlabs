# auto-improve

Auto-improve skills based on usage patterns.

## Execution

### Step 1: Analyze Usage
```
READ:
- .claude/analytics/skill-usage.yaml
- .claude/analytics/patterns-detected.yaml

IDENTIFY:
- Skills with low success rate
- Common errors
- Missing steps in execution
```

### Step 2: Identify Improvements
```
FOR EACH low-performing skill:
1. Analyze failure reasons
2. Identify missing steps
3. Generate improvement
4. Estimate impact
```

### Step 3: Apply Improvements
```
FOR EACH improvement:
1. Backup current skill
2. Update skill file
3. Log improvement
4. Monitor results
```

### Step 4: Verify
```
CHECK:
- Improvement applied successfully
- No regression
- Success rate improved
```

## Output
- Updated skills
- Improvement log
