# analyze-deps

Analyze project dependencies.

## Execution

### Step 1: Scan Dependencies
```
READ:
- go.mod
- package.json
- requirements.txt
- Cargo.toml
```

### Step 2: Analyze
```
CHECK:
- Outdated packages
- Security vulnerabilities
- Unused dependencies
- License issues
```

### Step 3: Recommendations
```
SUGGEST:
- Updates
- Alternatives
- Removals
- Security fixes
```

## Output
- Dependency report
- Recommendations
