# auto-monitoring

Auto-setup monitoring and observability.

## Execution

### Step 1: Detect Services
```
SCAN:
- Application endpoints
- Database connections
- External services
- Infrastructure
```

### Step 2: Create Dashboards
```
GENERATE:
- Application metrics
- Infrastructure metrics
- Business metrics
- Error tracking
```

### Step 3: Setup Alerts
```
CONFIGURE:
- Error rate thresholds
- Latency thresholds
- Resource usage alerts
- Custom alerts
```

### Step 4: Documentation
```
CREATE:
- Runbook
- Alert response guide
- Escalation procedures
```

## Output
- Grafana dashboards
- Alert configuration
- Runbooks
