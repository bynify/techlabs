# update-analytics

Update analytics with latest execution data.

## Execution

### Step 1: Collect Data
```
GATHER:
- Recent skill executions
- Agent task completions
- Decision outcomes
- Error logs
```

### Step 2: Update Files
```
UPDATE:
- .claude/analytics/skill-usage.yaml
- .claude/analytics/agent-performance.yaml
- .claude/analytics/decision-history.yaml
- .claude/analytics/patterns-detected.yaml
```

### Step 3: Detect Patterns
```
ANALYZE:
- User preferences
- Success patterns
- Anti-patterns
- Recommendations
```

### Step 4: Save
```
WRITE:
- Updated analytics files
- Pattern detection results
```

## Output
- Updated analytics
- Detected patterns
