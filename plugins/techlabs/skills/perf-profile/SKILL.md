# perf-profile

Profile application performance.

## Execution

### Step 1: Setup Profiling
```
CONFIGURE:
- Profiling tools
- Metrics to collect
- Duration
```

### Step 2: Run Profiling
```
EXECUTE:
- CPU profiling
- Memory profiling
- Database profiling
- Network profiling
```

### Step 3: Analyze
```
ANALYZE:
- Hot paths
- Memory leaks
- Slow queries
- Bottlenecks
```

### Step 4: Report
```
REPORT:
- Performance metrics
- Bottlenecks found
- Recommendations
```

## Output
- Profile report
- Bottleneck analysis
