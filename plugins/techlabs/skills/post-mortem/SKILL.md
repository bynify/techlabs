# post-mortem

Blameless post-mortem analysis.

## Execution

### Step 1: Document
```markdown
# Post-Mortem: [Incident Name]

## Summary
- Date: YYYY-MM-DD
- Duration: X hours
- Impact: Y users affected

## Timeline
- HH:MM - Event started
- HH:MM - Detected
- HH:MM - Mitigated
- HH:MM - Resolved

## Root Cause
[What caused it]

## Contributing Factors
- [Factor 1]
- [Factor 2]

## Action Items
- [ ] Prevention: [Action]
- [ ] Detection: [Action]
- [ ] Response: [Action]
```

## Output
- Post-mortem document
- Action items assigned
- Follow-up scheduled
