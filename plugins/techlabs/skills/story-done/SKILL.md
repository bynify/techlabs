# story-done

Verify story meets Definition of Done.

## Execution

### Step 1: Checklist
```markdown
## Story: [Title]

### Definition of Done
- [ ] Code complete
- [ ] Tests passing
- [ ] Code reviewed
- [ ] Documentation updated
- [ ] Acceptance criteria met
- [ ] Deployed to staging
```

### Step 2: Verify
```
STATUS: ✅ Done / ❌ Not Done

CHECKED:
- [x] Code complete
- [x] Tests passing
- [x] Code reviewed
- [ ] Documentation ← Missing
```

## Output
- Done status
- Missing items
- Action items
