# competitive-analysis

Analyze competitors' products, pricing, and positioning.

## When to Use
- Market research
- Feature planning
- Pricing decisions
- Positioning strategy

## Execution

### Step 1: Identify Competitors
```
ASK USER:
1. Direct competitors? (same solution)
2. Indirect competitors? (different approach)
3. Aspirational competitors? (market leaders)
```

### Step 2: Analysis Framework
```markdown
# Competitor: [Name]

## Overview
- Website: [url]
- Founded: [year]
- Funding: [amount]
- Employees: [count]

## Product
- Core features: [list]
- Pricing: [plans]
- Target audience: [who]

## Strengths
- [What they do well]

## Weaknesses
- [Where they fall short]

## Opportunities
- [Gaps we can exploit]
```

### Step 3: Feature Matrix
```
| Feature | Us | Competitor A | Competitor B |
|---------|----|--------------|--------------|
| Feature 1 | ✅ | ✅ | ❌ |
| Feature 2 | ✅ | ❌ | ✅ |
| Feature 3 | ❌ | ✅ | ✅ |
```

### Step 4: Positioning
```
DIFFERENTIATORS:
- [What makes us unique]
- [Our unfair advantage]
```

## Output
- Competitor profiles
- Feature matrix
- Positioning strategy
