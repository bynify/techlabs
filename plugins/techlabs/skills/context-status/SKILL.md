# context-status

Show current context window status.

## Execution

### Step 1: Analyze Context
```
CALCULATE:
- Total token usage
- Per-file breakdown
- Priority distribution
- Compression opportunities
```

### Step 2: Generate Report
```
DISPLAY:
- Context usage meter
- Top files by usage
- Priority breakdown
- Recommendations
```

## Output
- Context status report
- Recommendations
