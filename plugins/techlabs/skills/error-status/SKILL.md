# error-status

Show current error status and history.

## Execution

### Step 1: Load Error Data
```
READ:
- .claude/errors/error-log.yaml
- Recent error history
- Recovery success rates
```

### Step 2: Generate Report
```
DISPLAY:
- Recent errors (last 24h)
- Error patterns
- Recovery success rates
- Recommendations
```

## Output
- Error status report
- Recommendations
