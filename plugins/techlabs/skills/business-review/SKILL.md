# business-review

Complete business analysis: market, competition, pricing, metrics.

## When to Use
- Business planning
- Investor pitch
- Strategy review
- Market analysis

## What This Does

Chains multiple business skills:

```
FLOW:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: /market-research
  → Market analysis
  → TAM/SAM/SOM
  → Market trends

STEP 2: /competitive-analysis
  → Competitor profiles
  → SWOT analysis
  → Differentiation

STEP 3: /pricing-strategy
  → Pricing models
  → Competitive pricing
  → Value-based pricing

STEP 4: /jobs-to-be-done
  → User needs
  → Pain points
  → Value proposition

STEP 5: /churn-analysis
  → Retention metrics
  → Churn reasons
  → Retention strategies

FINAL: Business Report
  → Executive summary
  → Strategic recommendations
```

## Execution

### Step 1: Market Research
```
RUN /market-research

ANALYZE:
- Market size (TAM/SAM/SOM)
- Growth rate
- Trends
- Opportunities

CHECKPOINT:
"Market research complete. Continue to competition?"
```

### Step 2: Competitive Analysis
```
RUN /competitive-analysis

ANALYZE:
- Direct competitors
- Indirect competitors
- SWOT analysis
- Differentiation opportunities

CHECKPOINT:
"Competitive analysis complete. Continue to pricing?"
```

### Step 3: Pricing Strategy
```
RUN /pricing-strategy

ANALYZE:
- Pricing models
- Competitor pricing
- Value-based pricing
- Revenue projections

CHECKPOINT:
"Pricing strategy complete. Continue to user needs?"
```

### Step 4: Jobs to be Done
```
RUN /jobs-to-be-done

ANALYZE:
- User jobs
- Pain points
- Desired outcomes
- Value proposition

CHECKPOINT:
"User analysis complete. Continue to churn?"
```

### Step 5: Churn Analysis
```
RUN /churn-analysis

ANALYZE:
- Current churn rate
- Churn reasons
- Retention strategies
- LTV calculation

CHECKPOINT:
"Churn analysis complete. Generate business report?"
```

### Step 6: Generate Business Report
```markdown
# Business Review Report

## Date: 2024-01-15

## Executive Summary

### Market Opportunity
- TAM: $10B
- SAM: $2B
- SOM: $200M
- Growth: 15% YoY

### Competitive Position
- Direct competitors: 5
- Differentiation: [key differentiators]
- Competitive advantage: [advantages]

### Revenue Model
- Pricing: $29/month per user
- Expected ARPU: $35/month
- Projected MRR: $100K (Year 1)

### User Insights
- Primary job: [main job]
- Pain points: [top 3]
- Value proposition: [key value]

### Retention
- Target churn: <5% monthly
- Target LTV: $500
- Target CAC: $100

## Strategic Recommendations

### Short-term (1-3 months)
1. Focus on core features
2. Build MVP quickly
3. Get 10 beta users

### Medium-term (3-6 months)
1. Launch publicly
2. Achieve $10K MRR
3. Reduce churn to <5%

### Long-term (6-12 months)
1. Scale to $100K MRR
2. Expand to new markets
3. Hire team

## Financial Projections

| Metric | Month 3 | Month 6 | Month 12 |
|--------|---------|---------|----------|
| MRR | $5K | $25K | $100K |
| Users | 150 | 750 | 3000 |
| Churn | 8% | 5% | 3% |
| CAC | $50 | $40 | $30 |
| LTV | $200 | $350 | $500 |

## Risk Assessment

| Risk | Impact | Likelihood | Mitigation |
|------|--------|------------|------------|
| Competitor launches | High | Medium | Speed to market |
| Price sensitivity | Medium | High | Value communication |
| Churn | High | Medium | Onboarding optimization |
```
