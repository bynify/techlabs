# ui-review

UI consistency and quality review.

## Execution

### Step 1: Visual Consistency
```
CHECK:
- [ ] Consistent colors
- [ ] Consistent typography
- [ ] Consistent spacing
- [ ] Consistent components
```

### Step 2: UX Patterns
```
CHECK:
- [ ] Clear hierarchy
- [ ] Obvious CTAs
- [ ] Error states
- [ ] Loading states
- [ ] Empty states
```

### Step 3: Responsive
```
CHECK:
- [ ] Mobile (320px)
- [ ] Tablet (768px)
- [ ] Desktop (1024px)
- [ ] Large (1440px)
```

## Output
- Review report
- Issues list
- Recommendations
