# refactor-detect

Detect code smells and refactoring opportunities.

## Execution

### Step 1: Scan Codebase
```
ANALYZE:
- Function length
- Nesting depth
- Code duplication
- Complexity metrics
- Naming conventions
```

### Step 2: Identify Issues
```
DETECT:
- Long functions
- Deep nesting
- Duplicate code
- Complex logic
- Poor naming
```

### Step 3: Generate Report
```
REPORT:
- Issues found
- Severity ranking
- Refactoring suggestions
- Impact estimate
```

## Output
- Code smell report
- Refactoring suggestions
