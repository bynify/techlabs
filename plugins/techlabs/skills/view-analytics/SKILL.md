# view-analytics

View analytics dashboard for skill usage and agent performance.

## When to Use
- Analytics viewing
- Dashboard review
- Metric analysis
- Trend monitoring

## Execution

### Step 1: Load Analytics Data
```
READ:
- .claude/analytics/skill-usage.yaml
- .claude/analytics/agent-performance.yaml
- .claude/analytics/patterns-detected.yaml
```

### Step 2: Generate Dashboard
```
SHOW:
- Skill usage summary (most/least used)
- Agent performance metrics
- Success rates
- Common errors
- Detected patterns
```

### Step 3: Recommendations
```
SUGGEST:
- Skills that need improvement
- Agents that need training
- Patterns to reinforce
- Anti-patterns to fix
```

## Output
- Analytics dashboard
- Recommendations
