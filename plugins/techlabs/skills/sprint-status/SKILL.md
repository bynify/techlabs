# sprint-status

Sprint progress tracking and burndown.

## When to Use
- Sprint status
- Progress tracking
- Burndown charts
- Team velocity

## Execution

### Step 1: Status Report
```
SPRINT 5 - Day 7/10

COMPLETED: 22/34 points (65%)
IN PROGRESS: 8 points
TODO: 4 points

BURNDOWN:
Day 1: 34 pts
Day 3: 28 pts
Day 5: 24 pts
Day 7: 12 pts ← On track
```

### Step 2: Blockers
```
BLOCKED:
- Waiting for API credentials
- Design review pending

AT RISK:
- Export feature (complexity underestimated)
```

## Output
- Sprint status
- Burndown chart
- Blocker list
