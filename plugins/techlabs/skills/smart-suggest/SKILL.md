# smart-suggest

Provide smart suggestions based on context.

## Execution

### Step 1: Analyze Context
```
READ:
- Current task
- Project state
- User history
- Knowledge base
```

### Step 2: Generate Suggestions
```
SUGGEST:
- Relevant skills
- Similar past decisions
- Best practices
- Potential issues
```

### Step 3: Present
```
SHOW:
- Top 3 suggestions
- Confidence level
- Reasoning
```

## Output
- Suggestions
- Confidence scores
