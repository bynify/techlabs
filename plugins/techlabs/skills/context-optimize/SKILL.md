# context-optimize

Optimize context window for long sessions.

## Execution

### Step 1: Analyze Current Context
```
READ:
- Session state
- Active files
- Decision history
- User preferences

CALCULATE:
- Current token usage
- Priority ranking
- Compression potential
```

### Step 2: Optimize
```
OPTIMIZE:
1. Remove low-priority files
2. Summarize verbose sections
3. Compress old decisions
4. Preserve critical context

STRATEGY:
- High priority: Keep full
- Medium priority: Summarize
- Low priority: Remove
```

### Step 3: Update
```
UPDATE:
- Context config
- Session state
- Priority rankings
```

### Step 4: Report
```
SHOW:
- Context usage before/after
- Files optimized
- Recommendations
```

## Output
- Optimized context
- Usage report
