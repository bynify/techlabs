# user-guide

Create user guides for end users and administrators.

## When to Use
- New feature documentation
- User onboarding
- Help documentation
- Support materials

## Execution

### Step 1: Define Audience
```
AUDIENCE TYPES:
1. End Users - Basic features
2. Power Users - Advanced features
3. Administrators - System configuration
4. Developers - API integration
```

### Step 2: Create Guide Structure
```markdown
# User Guide

## Table of Contents
1. Getting Started
2. Basic Features
3. Advanced Features
4. Troubleshooting
5. FAQ

## Getting Started
### Step 1: Login
1. Go to https://app.example.com
2. Enter email
3. Enter password
4. Click "Sign In"

### Step 2: Dashboard
- Overview of key metrics
- Quick actions
- Recent activity
```

### Step 3: Write Content
```markdown
## Feature: User Management

### How to Add a User
1. Go to Settings → Users
2. Click "Add User"
3. Fill in the form:
   - Name: User's full name
   - Email: User's email address
   - Role: Select role
4. Click "Save"

### How to Edit a User
1. Go to Settings → Users
2. Find the user
3. Click "Edit"
4. Make changes
5. Click "Save"

### How to Delete a User
1. Go to Settings → Users
2. Find the user
3. Click "Delete"
4. Confirm deletion
```
