# milestone-review

Review project milestones and progress.

## Execution

### Step 1: Status Check
```
MILESTONE: [Name]
Status: In Progress | Complete | Blocked
Progress: 75%

COMPLETED:
- [x] Feature A
- [x] Feature B

REMAINING:
- [ ] Feature C
- [ ] Testing
```

### Step 2: Risks
```
BLOCKERS:
- Dependency on external API
- Resource availability

MITIGATIONS:
- Parallel development path
- Additional resources
```

## Output
- Milestone status
- Risk assessment
- Next steps
