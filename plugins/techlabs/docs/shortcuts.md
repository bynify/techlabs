## How to Use

1. Type the shortcut command in your chat
2. TechLabs will execute the workflow
3. Provide input when prompted
4. Review outputs at each checkpoint
5. Approve to continue

## Examples

```
/new-project "E-commerce platform for digital products"
/sprint-plan "User authentication sprint"
/audit-all
/production-deploy
```
